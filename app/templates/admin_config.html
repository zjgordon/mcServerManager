Configuration Help
Application Title

The title displayed in the browser tab and application header. This helps identify your server management instance.

Server Hostname/DNS

The DNS name or IP address used for server sharing links. Players will use this to connect: hostname:port

Total System Memory

This is the maximum amount of RAM that can be allocated across all Minecraft servers. Make sure this doesn't exceed your server's available RAM.

Maximum Per Server

This limits how much memory any single server can use. This prevents one user from consuming all available system memory.

Important Notes
  • Changes take effect immediately
  • Existing servers may need to be restarted
  • Ensure your system has enough RAM
  • Consider leaving some RAM for the OS
